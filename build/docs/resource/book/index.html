书籍

为什么要尽可能多读书（工具书、教科书、专业书）？ 因为作为普通人，根本没机会接触顶尖的大牛，就算公司里有，人家也根本不会搭理你。 读书是你接触顶级智者思想的唯一途径。

关于人​

控制饮食、减肥、锻炼身体、禁欲、推迟满足感、忍受孤独、戒除成瘾行为、无视他人评价、坚持、理性思考、脱离舒适区……这些都是反人性的事情。 我们在大多数情况下靠本能驱使，只不过大脑会让你有种幻觉，觉得这一切是你的意识自行决定的。
