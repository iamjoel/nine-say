# 书籍
> 为什么要尽可能多读书（工具书、教科书、专业书）？
> 因为作为普通人，根本没机会接触顶尖的大牛，就算公司里有，人家也根本不会搭理你。
> 读书是你接触顶级智者思想的唯一途径。

> 我们应该努力意识到，我们很容易受到模式的诱惑。我们不应该受到引诱，应该保持怀疑的态度。
